Much of machine learning is statistics-based, and so certain well-known classsifiers/regressors (such as naive Bayes) require some knowledge of probability theory. Also, we need to know something about statistical tests in order to evaluate the models we create.

Many of the examples will utilize the SciPy, which is a collection of math- and science-related libraries.

Probability Distributions¶

For arbitrary sets of numerical values, the mean, median, standard-deviation, etc., are easy to understand and compute, provided the sets are finite. When dealing with infinite sets, things are a bit less intuitive, but these concepts still make sense.

Consider an infinite sequence of rolls of a 6-sided die (suppose the die is not fair, so that 4 comes up more often). A sample is a (potentially infinite) subset of that sequence, e.g., $6, 3, 5, 1, 2, 2, 4, 5, 6, \ldots$. Alternatively, consider taking measurements of a physical, real-valued phenomenon. We can only collect a finite sample of measurements, but in theory at least there are an infinite number of measurements we could make. In such cases, we talk about random variables, expected values, and probability distributions.

Random Variables and Probability Distributions¶

A random variable is a variable whose value is subject to chance.

  • Discrete random variables take values from a finite or countably infinite set.

  • Continuous random variables take values from some real-valued interval.

The set of possible values is called the sample space, symbolized $S$, $\Omega$, or $U$. Below, we'll use $S$.

  • A single item from $S$ is called an outcome.
  • A subset of $S$ is called an event.

Example:

  • Let $S = \{1,2,3,4,5,6\}$.
  • Event 1 = $\{2,4,6\}$ (rolling an even number)
  • Event 2 = $\{1,5\}$ (rolling a 1 or a 5)

The probability of an event is the proportion of times that the event would occur based on a repeated random sampling of the variable. E.g., the probability that a 1 is rolled ($Pr(X = 1)$) is 1/6th, while the probability of rolling a 5 or less ($Pr(X \leq 5)$) is 5/6th.

A probability distribution for random variable $X$ is a function from values of $X$, or a range of values of $X$, to the interval $[0,1]$. There are an infinite number of possible distributions. In a uniform distribution, each outcome is equally likely. Rolls of a fair die are uniformly distributed, while those of a die that favors one side are not.

If $X$ is discrete, a probability mass function (pmf) $f$ is used to describe the distribution: $f$ is any function such that for each $e\in S$,

  • $0 \leq f(e) \leq 1$, and
  • $\displaystyle\sum_{e\in S} f(e) = 1$

In the discrete case, $f(e)$ is the probability that $X$ takes value $e$. That is, $f(e)$ is $Pr(X = e)$.

If $X$ is instead continuous, a probability density function (described below) is used.

Expected Value (Mean) and Variance of Discrete Distributions¶

For a discrete random variable $X$, the average, mean, or expected value $\mu$ (or $E[X]$) is defined as follows:

$\mu =\displaystyle\sum_{e \in X}e\times Pr(e)$

As we continue to roll a fair 6-sided die, the mean value of the growing sample (a finite set) will converge to $\mu$.

$1*1/6 + 2*1/6 + 3*1/6 + 4*1/6 + 5*1/6 + 6*1/6 = 21/6= 3.5$

The variance $\sigma^2$ for a discrete random variable is:

$\sigma^{2} =\displaystyle\sum_{e\in X}(e - \mu)^{2}Pr(e) = E[(X - \mu)^{2}]$

For a fair die:

$\sigma^{2} = (1-3.5)^{2}*\frac{1}{6} + (2-3.5)^{2}*\frac{1}{6} + (3-3.5)^{2}*\frac{1}{6} + (4-3.5)^{2}*\frac{1}{6} + (5-3.5)^{2}*\frac{1}{6} + (6-3.5)^{2}*\frac{1}{6}$

$\sigma^{2} = \frac{1}{6}[(-2.5)^{2}+ (-1.5)^{2} + (-0.5)^{2} + (0.5)^{2} + (1.5)^{2} + (2.5)^{2}]$

$\sigma^{2} = \frac{1}{6}\times 17.5 \approx2.916667$

The standard deviation, $\sigma$, is the square-root of the variance.

Cumulative Distribution Functions¶

If $X$ is numeric, a cumulative distribution function cdf can be defined based on the probability mass function.

$cdf(a) =\displaystyle\sum_{x\leq a}Pr(x)$

$cdf(a)$ computes the probability that a arbitrarily selected value from $X$ is $\leq a$. We can write this $Pr(Xa)$. Observe that this is a monotone non-decreasing function.

In SciPy, cdf can be used to calculate $Pr(X \leq a)$ for a given $a$.

Probability Density Functions¶

For continuous random variables, the probability of a single outcome is 0, and so we don't use a probability mass function as the distribution. Instead we use a probability density function $f$.

The function $f$ is typically defined over the real numbers $\mathbf{R}$; it should have the following properties.

  • For each $x\in \mathbf{R}$, $f(x) \geq 0$.
  • $\displaystyle\int_{-\infty}^{\infty} f(x)dx = 1$. That is, the area under the curve must be 1.

The probability that $X$ is between $a$ and $b$ is defined as: $Pr(a\leq x \leq b) = \displaystyle\int_{a}^{b} f(x)dx$

For a continuous random variable $X$, the mean is:

$\displaystyle\int_{-\infty}^{\infty} x f(x)dx$.

The variance of a continuous random variable is:

$ \sigma^{2} = \displaystyle\int_{-\infty}^{\infty} (x-\mu)^{2}f(x)dx = E[(X - \mu)^{2}]$

Finally, the cumulative distribution function based on the pdf $f$ is:

$ cdf(b) = Pr(X\leq b) = \displaystyle\int_{-\infty}^{b} f(x)dx$

Again, this is a monotone non-decreasing function.

In SciPy, we can use rvs from scipy.stats.uniform to generate uniformly distributed floating point values.

Normal Distributions¶

Of special note are so-called normal distributions $N(\mu,\sigma)$, defined by probability distribution functions having the below form, where $\mu$ is the mean, and $\sigma$ is the standard deviation.

$f(x) = \frac{1}{\sigma\sqrt{2\pi}} ^{\frac{(x-\mu)^{2}}{2\sigma^{2}}}$

Each function will generate a symmetric, bell-shaped curve. The standard normal distribution has $\mu =0$ and $\sigma = 1$.

In [9]:
import matplotlib
import matplotlib.pyplot as plt
import math

def y(x, mu=0, sigma=1):
    return 1/(sigma*math.sqrt(2*math.pi))*math.e**(-1*((x-mu)*(x-mu))/(2*sigma*sigma))
x = np.linspace(-4,4,200)

plt.plot(x,y(x))
plt.plot(x,y(x,0,1.25))
plt.plot(x,y(x,0,2))
pass

Increasing $\mu$ widens the curve. (In)decreasing $\sigma$ shifts the curve.

In SciPy, we can use norm from scipy.stats to create normal distributions (normally distributed variables). The attributes loc and scale are used to set the mean and standard deviation, respectively. The Empirical Rule¶

If $X$ is normally distributed, then the following all hold.

  • $Pr(\mu-1\sigma \leq x \leq \mu+1\sigma) \approx 68\%$
  • $Pr(\mu-1.65\sigma \leq x \leq \mu+1.65\sigma) \approx 90\%$
  • $Pr(\mu-1.96\sigma \leq x \leq \mu+1.96\sigma) \approx 95\%$
  • $Pr(\mu-3\sigma \leq x \leq \mu+3\sigma) \approx 100\%$

In other words, roughly 95% of values are within 2 standard deviations of the mean.

Standardization

For a normal distribution with $\mu \neq 0$ and/or $\sigma \neq 1$, we can convert values of the random variable to a corresponding z-score. Recall, $z$ denotes the number of standard deviations from the mean. So, given an $x$ taken from $N(\mu,\sigma)$, the distance between $x$ and $\mu$ is given by $x-\mu = z\sigma$. Solving for $z$ yields

$= \frac{(\mu)}{\sigma}$

So, if we know $\mu$ and $\sigma$, to compute a z-score for $x$, we subtract $\mu$ from $x$ and then divide by $\sigma$

Example:

Given distribution $N(\mu=3.14,\sigma=2.5)$, what is $Pr(x \leq 4)$?

Using the above equation, we find the z-score for $x=4$.

$z = (4-3.14)/2.5 = 0.344$

We can then use a z-table such as the one linked to above to determine that $Pr(x \leq 4)$ is in the range (63.307%, 63.3683%).

The Central Limit Theorem¶

Suppose some population $P$ has mean $\mu$ and standard deviation $\sigma$. Now consider the set of random samples taken from $P$, each with size $n$. Each sample has a measurable mean and standard deviation.

Given the above, the following all hold.

  • The means of the samples will be approximately normally distributed.
  • The mean of the sample means is $\mu$ (the mean of $P$).
  • The standard deviation of the sample means is $\frac{\sigma}{\sqrt{n}}$.

Note that the original population $P$ need not be normally distributed.

The term $\frac{\sigma}{\sqrt{n}}$ is called the standard error (of the mean, sometimes written SEM). Note that as the standard deviation of the sample means, it is a measure of dispersion.

Given that $n$ is in the denominator, as $n$ gets bigger, $\frac{\sigma}{\sqrt{n}}$ gets smaller. That is, as the sample size grows larger, the narrower the distribution of sample means will be.

Testing the Theorem¶

We can run tests to help convince us that the Central Limit Theorem is true. Below, we create a function that accepts a population and a sample size and then generates multiple samples of the given size. The mean and standard deviation of the sample means is then calculated.

We'll use the function to test what happens as we change the sample size.

In [18]:
import numpy as np
import matplotlib
import matplotlib.pyplot as plt

def clt_test (sample_size, pop, iterations=1000):
    sample_means = np.empty(iterations);
    pos = 0; 
    for it in range(iterations):
        s = np.random.choice(pop, size=sample_size, replace=False)
        sample_means[pos] = np.mean(s)
        pos += 1

    mean_sample_means = np.mean(sample_means)
    sd_sample_means = np.std(sample_means)
    return sample_means

def write_results (sample_size, pop, sample_means):
    pop_mean = np.mean(pop)
    pop_sd = np.std(pop)
    standard_error = pop_sd/np.sqrt(sample_size)
    mean_sample_means = np.mean(sample_means)
    sd_sample_means = np.std(sample_means)
    print("population mean: ", pop_mean)
    print("population sd: ", pop_sd)
    print("standard_error: ", standard_error)
    print("mean of sample means: ", np.mean(sample_means))
    print("sd of sample means: ", sd_sample_means)

A Preview of Confidence Intervals¶

As the sample size increases, the standard deviation of the sample means (i.e., the standard error) gets smaller (the sample means won't be very spread out).

Suppose we use a large sample size $n$. Because of this, the mean of a randomly selected sample will very likely be close to the other sample means and also to the true population mean $\mu$. More precisely, if we know the standard error $\frac{\sigma}{\sqrt{n}}$, since the distribution of means is approximately normally distributed with mean $\mu$ and standard deviation $\frac{\sigma}{\sqrt{n}}$, we can calculate for any sample $s_i$ the probability that $\mu$ (the actual population mean) is within a desired range.

Recall that for any normal distributed $P$ with mean $\mu_P$ and standard deviation $\sigma_P$, $Pr(\mu_P-1.65\sigma_P \leq x \leq \mu_P+1.65\sigma_P) \approx 90\%$.

When speaking of the means of samples, we use $\frac{\sigma}{\sqrt{n}}$ as the standard deviation. So, if $m$ is the mean of a sample, then 90% of the time, $\mu$ is $m \pm 1.65 \frac{\sigma}{\sqrt{n}}$.

  • 90% of the time, $m$ is $\leq 1.65 \frac{\sigma}{\sqrt{n}}$ units away from $\mu$.

  • 90% of the time, $\mu$ is $\leq 1.65 \frac{\sigma}{\sqrt{n}}$ units away from $m$.

  • 90% of the time, $\mu$ is $\leq m \pm 1.65 \frac{\sigma}{\sqrt{n}}$.

That is, if we knew $\frac{\sigma}{\sqrt{n}}$, then given a particular sample, we could calculate a confidence interval for the true mean of the population. Above, we've calculated the 90% confidence interval, but other levels could be used.

However, note that calculating the confidence interval requires knowing $\sigma$, the true population standard deviation. We typically don't know that any more than we know the true population mean $\mu$. Because of this, in order to calculate confidence intervals, the sample standard deviation is used as a substitute, and also another distribution (a t-distribution rather than a normal distribution) is assumed.

This is covered in another section.
